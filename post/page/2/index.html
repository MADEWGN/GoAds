Apa itu AMP ? AMP (Accelerated Mobile Pages) atau sering disebut dengan AMP HTML merupakan salah satu project yang dikembangkan oleh Google. AMP HTML memiliki struktur yang meminimalisir penggunaan JavaScript, CSS, HTML. Jadi ada aturan tertentu dalam penulisan tag atau baris kode pada website AMP HTML. Tujuan dari AMP HTML yaitu untuk meningkatkan kecepatan akses sebuah website, khususnya akses website dari device smartphone. Jadi untuk teman-teman yang mungkin pengunjung situsnya kebanyakan dari smartphone, menggunakan AMP HTML adalah salah satu alternatif untuk meningkatkan user experience pada situs Anda. [Read More]
